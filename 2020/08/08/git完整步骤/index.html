git完整步骤

发布于 2020-08-08 评论

Git 相关笔记，方便查询

拉取和提交

  1. 现在本地初始化一个厂库：git init

  2. 配置好远程url地址：git remote add origin git@XX.XXX.XXX.XX:test/pytest.git

  3. 直接拉代码：git pull origin branch

  4. 查看代码修改状态：git status

  5. 创建一个你要上传的分支（会同时切换到该分支）：git checkout -b branch

  6. 先提交到本地分支：git add .

  7. 提交说明：git commit -m “修改代码说明”

  8. 再推送到远程分支：git push -u origin branch

fatal: refusing to merge unrelated histories

强制合并

git pull origin master --allow-unrelated-histories

git 中.gitignore文件不生效

git rm -r --cached .
git add .
git commit -m 'update .gitignore'
