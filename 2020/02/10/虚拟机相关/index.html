虚拟机相关

本地主机访问虚拟机的web服务器

  1. 首先虚拟机处进行设置，网络模式为NAT

  2. 需要将服务器的地址改为公网地址0.0.0.0:8111。

    直接在windows浏览器里url输入”192.168.X.X:8111”,其中192.168.X.X是你虚拟机的ip，这样就可以访问了。

python manage.py runserver 0.0.0.0:8111
#主机
http://192.168.111.131:8111/

虚拟机屏幕太小

Vmware Tools工具官方已经不建议。更提倡开源的open-vm-tools

第一步： 安装开源软件 sudo apt-get install open-vm-tools

第二步 ： sudo apt-get install open-vm* ，安装后直接打开全屏即可，若屏幕没有改变则重启一下即可。

重启命令：reboot

ssh连接本地虚拟机

查看本地ip地址

ifconfig ping
apt-get install net-tools

Ubuntu远程登录服务器——SSH的安装和配置

  1. 最好先更新一下 apt-get
    sudo apt-get update

  2. 一般ssh功能是linux系统默认有安装的，如果没有的话，就安装一下

     sudo apt install openssh-client

  3. 安装ssh

     sudo apt install openssh-server

  4. 查看ssh服务是否启动

     sudo ps -e |grep ssh
