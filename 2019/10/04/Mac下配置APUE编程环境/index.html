Mac下配置APUE编程环境

下载源代码

cd apue.3e #进入目录

同时修改include/apue.h文件 130行，在#endif /* _APUE_H */之前加上一条语句

#include "error.c"

void    WAIT_CHILD(void);
#include "error.c"
#endif  /* _APUE_H */

之后编译

make
cp ./include/apue.h /usr/local/include/
cp ./lib/error.c /usr/local/include/

如果之前编译了，就到/usr/local/include/修改也可以
