头插法

public ListNode reverseList(ListNode head) {
        ListNode ret = new ListNode(-1);
        ListNode tmp = null;
        while(head != null){
            tmp = head.next;
            head.next = ret.next;
            ret.next = head;
            head = tmp;
            tmp = null;
        }
        return ret.next;        
}

使用临时链表，首先保存head.next ，然后head.next指向返回链表ret.next，让其暂时清空尾部无用节点，返回链表ret.next再指向head，即头部加入新的节点，然后head再指向tmp保存的head.next，tmp清空。

尾插法

public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
    // 类似归并排序中的合并过程
    ListNode dummyHead = new ListNode(0);
    ListNode cur = dummyHead;
    while (l1 != null && l2 != null) {
        if (l1.val < l2.val) {
            cur.next = l1;
            cur = cur.next;
            l1 = l1.next;
        } else {
            cur.next = l2;
            cur = cur.next;
            l2 = l2.next;
        }
    }
    // 任一为空，直接连接另一条链表
    if (l1 == null) {
        cur.next = l2;
    } else {
        cur.next = l1;
    }
    return dummyHead.next;
}

新建dummyHead，再将新建的cur指向dummyHead，之后不管dummyHead，只使用cur.next来控制就可以了，最后返回dummyHead。

注意：cur = cur.next，等于对cur自己的重定向，cur.next = cur.next.next等于对整个的链表重定向。

Solution {
    public TreeNode invertTree(TreeNode root) {
        if (root == null) {//一定要检查指针为空
            return null;
        }
        root.left = invertTree(root.left);
        root.right = invertTree(root.right);

        TreeNode tmp = root.left;
        root.left = root.right;
        root.right = tmp;
        return root;
    }
}

// 得到二叉树的镜像 —— 不使用递归
    public void MirrorNotRecursive(TreeNode root) {
        LinkedList stack = new LinkedList();
        TreeNode temp = null;
        if(root == null) {
            return;
        }
        stack.add(root);
        while(stack.size() != 0) {
            TreeNode node = stack.removeFirst();
            temp = node.left;
            node.left = node.right;
            node.right = temp;
            if(node.right != null) {
                stack.add(node.right);
            }
            if(node.left != null) {
                stack.add(node.left);
            }
        }
    }
