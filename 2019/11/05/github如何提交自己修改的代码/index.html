github 提交自己修改的代码

发布于 2019-11-05 评论 杂项  技巧 

步骤:

  1. fork 一份代码到自己的仓库

    进入 github 要修改项目的主页，点击 fork 按钮，fork 一份代码到自己的仓库

  2. clone 出 fork 后的项目

    git clone https://github.com/xxxx/xxxx.git

  3. 使本地代码与原始项目关联

    git remote add upstream https://github.com/xxxx/xxxx.git

  4. 关联后，拉取原始项目代码到本地

    git fetch upstream

  5. 本地进行修改

  6. 提交代码到自己 fork 的 github 仓库

    git commit -a -m “修改内容”

    git push

  7. 进入 github 上自己 fork 的项目，创建提交请求到原始代码

    • New pull request
    • Create pull request

  8. 之后等待原始项目的拥有者 merge 你提交的代码
